## 📋 使用指南

### 1. 启动导航流程

```typescript
// 步骤1: 从首页进入目的地选择页
router.pushUrl({ url: 'pages/DestinationSelectPage' });

// 步骤2: 选择目的地后跳转到路径概览页
router.pushUrl({
  url: 'pages/NavigationSummaryPage',
  params: {
    destinationId: 'DEPT_INTERNAL_301',
    destinationName: '内科诊区301'
  }
});

// 步骤3: 点击"开始导航"跳转到AR导航页
router.pushUrl({
  url: 'pages/ARNavigationPage',
  params: {
    destinationId: 'DEPT_INTERNAL_301',
    destinationName: '内科诊区301',
    pathData: JSON.stringify(navigationPath)
  }
});
```

### 2. 编程方式启动导航

```typescript
import { ARNavigationService } from '../ar/ARNavigationService';
import { PathPlanner } from '../ar/PathPlanner';

// 初始化
const pathPlanner = PathPlanner.getInstance();
await pathPlanner.initialize(hospitalData);

const navService = ARNavigationService.getInstance();

// 启动导航
await navService.startNavigation(destination, {
  mode: NavigationMode.AR_MODE,
  voiceEnabled: true,
  accessibilityMode: false
});

// 监听事件
navService.addEventListener(NavigationEventType.ARRIVED, (event) => {
  console.log('已到达目的地');
});
```
