IDE 编译错误快速修复
===================================

错误: java: 程序包com.example.medical.common不存在

【立即执行】
1. 打开 IntelliJ IDEA
2. 右侧 Maven 工具栏 → 点击刷新按钮 (Reload All Maven Projects)
3. Build → Rebuild Project

【如果仍然失败】
File → Invalidate Caches → Invalidate and Restart

【JDK配置】
File → Project Structure → Project → SDK: 选择 JDK 1.8

【验证】
common包文件路径: src/main/java/com/example/medical/common/
- BCryptUtil.java ✓
- JwtUtil.java ✓
