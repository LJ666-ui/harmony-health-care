# 医院分类优化方案

## 一、问题分析

### 原有分类问题
前端定义的医院分类与数据库实际情况不符：
- **前端分类**：三级甲等、三级乙等、二级甲等、二级乙等、一级甲等、一级乙等、综合医院、专科医院
- **数据库实际**：只有三级甲等、综合医院、社区医院、专科医院、卫生院、保健机构、诊所

### 问题影响
1. 用户点击"三级乙等"、"二级甲等"等不存在的分类，筛选结果为空
2. 数据库中实际存在的"卫生院"、"保健机构"、"诊所"等分类无法筛选
3. 用户体验差，筛选功能不完整

---

## 二、优化方案

### 1. 数据库重新分类

**执行脚本**：`sql/reorganize_hospital_category.sql`

**分类逻辑**：
```sql
-- 基于department字段和name字段综合判断
1. 三级甲等：department = '三级甲等医院'
2. 综合医院：department = '综合医院'
3. 专科医院：department = '专科医院' 或包含中医院、妇幼等关键词
4. 卫生院：department = '卫生院'
5. 社区医院：社区卫生服务中心/站
6. 保健机构：department = '医疗保健服务场所'
7. 诊所：department = '诊所'
```

**分类结果**：
| 分类 | 数量 | 说明 |
|------|------|------|
| 综合医院 | 121家 | 各类综合医院、人民医院 |
| 三级甲等 | 116家 | 三甲医院 |
| 社区医院 | 87家 | 社区卫生服务中心/站 |
| 专科医院 | 25家 | 中医院、妇幼医院等 |
| 保健机构 | 18家 | 医疗保健服务场所 |
| 诊所 | 9家 | 各类诊所 |
| 卫生院 | 5家 | 乡镇卫生院 |
| **总计** | **381家** | 全部医院 |
