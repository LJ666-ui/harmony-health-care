## 🚀 部署指南

### 1. 数据库部署

```bash
mysql -u root -p
source update-database-optimization.sql
source verify-database-update.sql
```

### 2. 后端部署

```bash
cd e:\HMOS6.0\Github\harmony-health-care
mvn clean package
mvn spring-boot:run
```

访问：`http://localhost:8080/swagger-ui.html`

### 3. 前端部署

```bash
hvigorw assembleHap
hdc install entry-default-signed.hap
```

### 4. 测试

```bash
mvn test
```
