```

        ↓

         ↓
┌─────────────────────┐

│ ┌─────────────────┐ │
│ │ConversationMemory│ ← 10轮对话记忆
│ └─────────────────┘ │
│ ┌─────────────────┐ │
│ │RAG Retriever    │ ← 知识库检索增强
│ │ ├─向量检索       │ │
│ │ ├─关键词检索     │ │
│ │ └─重排序         │ │
│ └─────────────────┘ │
│ ┌─────────────────┐ │
│ │Specialist Agents│ ← 专科智能体
│ │ ├─神经内科       │ │
│ │ ├─眼科           │ │
│ │ └─其他5个专科... │ │
│ └─────────────────┘ │
└────────┬────────────┘
         ↓
   综合会诊报告输出
```

### 💪 核心能力

- ✅ **意图识别**: 自动分类7种问诊意图

- ✅ **多轮对话**: 维护10轮上下文记忆
- ✅ **专科协作**: 7个专科智能体联合会诊

### 古籍医典图像复原 (8阶段流水线)

```
原始古籍图像
    ↓ [1] 图像预处理 (去噪、增强)
    ↓ [2] 超分辨率重建 (4-8倍提升)
    ↓ [3] AI色彩还原 (褪色恢复)
    ↓ [4] OCR古文字识别 (繁体/竖排)
    ↓ [5] 现代医学标注
    ↓ [6] 智能破损修复 (inpainting)
    ↓ [7] 处理报告生成
    ↓ [8] 高清数字化输出 ✓
```

**价值**: 保护文化遗产 + 现代化利用

### 目标用户群体

|---------|---------|--------------|

✅ **看病难**: 在线预约+AR导航，就医流程数字化  
✅ **健康管理难**: AI风险评估+个性化建议  
✅ **数据孤岛**: 分布式协同打破设备壁垒  
✅ **适老化不足**: 全链路无障碍设计  
✅ **效率低下**: 智慧病房自动化提效
