### 👆 点什么、测什么

进入后看到聊天界面 + 6个快捷问题：

```
┌─────────────────────────────────────┐
│  您好! 我是AI健康顾问...             │
│                                     │
│  ❤ 血压多少算正常?    ← 点这个      │ 测试DeepSeek基础问答
│  🩺 老年人如何预防糖尿病? ← 点这个  │ 
│  🏃 每天应该运动多久?   ← 点这个  │ 
│  😵 失眠怎么办?         ← 点这个  │ 测试DeepSeek复杂场景
│  💪 如何保持心脏健康?   ← 点这个  │ （症状/用药/心理等）
│  🥗 饮食上有什么注意事项? ← 点这个 │ 
│                                     │
│  ┌──────────────────────┐          │
│  │ 输入问题...          │ ← 或自己输入 │ 测试自由对话+多轮
│  └──────────────────────┘          │
│            [发送]                   │
└─────────────────────────────────────┘
```

### ✅ 推荐测试顺序

#### 第1步：点6个预设问题（每题等2-3秒）
1. ❤️ **血压多少算正常?**
   - 预期：数值范围 + 高血压分级标准
   
2. 🩺 **老年人如何预防糖尿病?**
   - 预期：饮食/运动/筛查/体重管理建议
   
3. 🏃 **每天应该运动多久?**
   - 预期：150分钟/周 + 有氧+力量结合
   
4. 😵 **失眠怎么办?**
   - 预期：原因分析 + 非药物方法 + 就医时机
   
5. 💪 **如何保持心脏健康?**
   - 预期：七大建议（饮食/运动/戒烟/减压等）
   
6. 🥗 **饮食上有什么注意事项?**
   - 预期：营养素分类 + 具体食物推荐

#### 第2步：自由输入测试多轮对话

在输入框输入：

```
第1轮: 我有高血压，能喝咖啡吗？
        ↓ 等回复（应提到适量/脱因咖啡因）
        
第2轮: 那茶呢？绿茶可以吗？
        ↓ 等回复（应结合上下文，提到与咖啡对比）
        
第3轮: 还有其他饮料推荐吗？
        ↓ 等回复（应总结前两轮，给出综合建议）
```

**验证要点**：
- ✅ AI理解上下文（知道你在问"除了咖啡和茶"）
- ✅ 回复连贯不矛盾
- ✅ 每轮响应1-3秒
