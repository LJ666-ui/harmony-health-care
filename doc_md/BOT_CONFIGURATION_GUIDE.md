### 推荐的开场白预制文案

```
您好！我是您的专属健康咨询助手。

我可以为您提供以下专业咨询：

🩺 内科健康咨询
   • 血压、血糖、心脏等慢性病管理

   • 常见症状分析和就医指导

👁️ 眼科保健指导
   • 视力问题（老花眼、白内障、青光眼）
   • 眼部护理和日常保健
   • 眼科手术咨询和术后护理

💪 康复训练指导
   • 术后康复计划制定
   • 跌倒预防和平衡训练
   • 功能恢复和辅助器具使用

━━━━━━━━━━━━━━━━━━━━━━━

💡 如何使用：
   直接告诉我您的健康问题，例如：
   • "血压高怎么办"
   • "白内障什么时候手术好"
   • "骨折术后如何康复"

⚠️ 重要提醒：
   我是健康咨询师，不是医生。
   我的建议仅供参考，不能替代医生的诊断和治疗。
   如有紧急情况，请立即就医！
```

## 5. 对话管理配置

### 位置

左侧面板 → 「对话管理」或「会话设置」区域

### 推荐配置

#### 5.1 会话设置

| 配置项 | 推荐设置 | 说明 |
|-------|---------|------|
| 单轮最大消息数 | 20 | 防止异常长对话消耗Token |
| 未识别回复次数上限 | 3 | 超过后引导用户使用快捷指令 |
| 会话超时时间 | 30分钟 | 超过后自动结束会话 |
| 历史消息记忆轮数 | 10 | 保持上下文连贯性 |

#### 5.2 对话开场设置

- 开启「用户首条消息快速匹配」
- 开启「意图分类优先匹配快捷指令」
- 关闭「默认发送问候语」（已配置开场白）

#### 5.3 敏感问题处理

当用户提问涉及以下内容时，返回预设回复：

| 敏感类型 | 回复内容 |
|---------|---------|
| 自杀/自残 |抱歉，听到您有这样的想法，我非常担心。请您立即拨打心理援助热线：400-821-1212，寻求专业帮助。您不是一个人，有很多人愿意帮助您。|
| 涉暴/违法 |抱歉，我无法协助处理此类请求。如果您遇到法律问题，建议您咨询专业律师或拨打法律服务热线。|
| 医疗紧急情况 |⚠️如果您或他人正在经历紧急医疗状况，请立即拨打120急救电话或前往最近的医院急诊就诊。|

#### 5.4 情绪识别与应对

| 识别结果 | 应对策略 |
|---------|---------|
| 用户表达焦虑/担忧 |先给予情感支持，再提供专业建议 |
| 用户表达感谢 |简洁回应并引导继续提问 |
| 用户表达不满 |道歉并尝试重新理解需求 |
| 用户反复询问同一问题 |确认是否需要更详细的解释 |

## 6. 安全与权限设置

### 位置

左侧面板 → 「安全与权限」或「权限设置」区域

### 6.1 内容安全配置

#### 安全过滤级别

| 过滤项 | 推荐级别 | 说明 |
|-------|---------|------|
| 敏感词过滤 | 中等 | 过滤明显违规内容，保留正常医疗术语 |
| 政治敏感内容 | 严格 | 一旦触发直接拒绝回答 |
| 医疗建议边界 | 中等 | 区分健康咨询与医疗诊断 |

#### 医疗场景特殊规则

- 禁止提供具体处方建议
- 禁止推荐特定药品名称（可提供药品类别）
- 禁止预测具体疾病诊断
- 所有医疗建议必须包含「请咨询医生」提示

### 6.2 用户数据保护

| 配置项 | 推荐设置 |
|-------|---------|
| 对话记录保存 | 开启（仅用于服务质量改进） |
| 用户信息采集 | 关闭 |
| 第三方数据共享 | 关闭 |
| 医疗数据加密 | 强制开启 |

### 6.3 访问权限配置

| 配置项 | 推荐设置 |
|-------|---------|
| Bot访问方式 | 公开（需审核） |
| 每日调用上限 | 1000次/天 |
| 单用户调用频率 | 60次/分钟 |
| 商用授权 | 需单独申请 |

### 6.4 审计与日志

| 配置项 | 推荐设置 |
|-------|---------|
| 操作日志 | 开启 |
| 异常行为告警 | 开启 |
| 调用量统计 | 开启 |
| 日志保存周期 | 30天 |

### 6.5 敏感操作二次确认

开启以下操作的二次确认：

- 退出对话
- 清除历史记录
- 切换咨询类型

### 发布前检查

- [ ] 所有快捷指令配置完成

### 发布流程

1. 点击右上角「发布」
2. 选择发布渠道（抖音/微信/网站等）
3. 阅读并同意平台协议
4. 提交审核（通常1-3个工作日）
5. 审核通过后正式上线

### 发布后监控

- 关注用户反馈和评价
- 定期查看调用量统计
- 监控异常调用告警
- 根据用户反馈优化快捷指令
