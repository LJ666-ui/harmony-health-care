# 小艺3个Plugin - 直接复制配置清单

> 直接复制下面的内容，粘贴到华为HAG平台的插件配置页

---

## Plugin 1: 健康风险评估

```
【基本信息】
插件名称: 健康风险评估
插件描述: 基于MindSpore Lite引擎，评估用户健康风险，包括高血压、糖尿病、跌倒、衰弱、肌少症5个维度

【Intent配置】
Intent名称: CHECK_HEALTH_RISK
Intent描述: 用户请求进行健康风险评估

【训练话语】
评估我的健康风险
我的健康状况怎么样
风险评估
健康检查
帮我看看有没有健康问题
查看我的健康报告
我有哪些健康风险
我的身体怎么样
```

---

## Plugin 2: 医疗图像识别

```
【基本信息】
插件名称: 医疗图像识别
插件描述: 基于HiAI NPU引擎，识别医疗图像，包括皮肤病变、药品识别、眼底照片分析、跌倒姿态检测

【Intent配置】
Intent名称: RECOGNIZE_MEDICAL_IMAGE
Intent描述: 用户上传图片请求识别

【训练话语】
识别这张图片
拍照识别
这是什么
帮我看看这张照片
皮肤有问题帮我看看
药品识别
检测有没有跌倒
眼底照片分析
这是什么药
帮我看看有没有皮肤病
帮我识别一下
```

## 平台操作步骤

```
1. 登录华为HAG平台
2. 进入"智慧康养助手"大智能体
3. 左侧菜单点"插件"
4. 点"添加插件"
5. 选择"创建自定义插件"
6. 选第1个Plugin，照着上面的内容填写
7. 保存
8. 重复4-7，填第2、3个Plugin
```
