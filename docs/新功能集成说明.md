## 一、已集成的功能

### 1. RAG检索系统

**位置**: `entry/src/main/ets/rag/`

**文件列表**:
- `HybridRetriever.ets` - 混合检索器（向量+关键词）
- `VectorRetriever.ets` - 向量检索器（余弦相似度）
- `KeywordRetriever.ets` - 关键词检索器（BM25算法）
- `Reranker.ets` - 重排序器（Cross-Encoder）
- `index.ets` - 统一导出

**集成到AI Agent**:
- `EnhancedRAGRetriever.ets` - 增强版RAG检索器，已集成到AI Agent系统

**使用方法**:
```typescript
import { HybridRetriever } from '../rag';

const retriever = new HybridRetriever({
  vectorWeight: 0.6,
  keywordWeight: 0.4,
  useRerank: true,
  topK: 10
});

const results = await retriever.retrieve('高血压怎么治疗');
```

### 2. 无障碍组件库

**位置**: `entry/src/main/ets/components/`

**文件列表**:
- `accessible/AccessibleButton.ets` - 无障碍按钮
- `accessible/AccessibleText.ets` - 无障碍文本
- `accessible/AccessibleInput.ets` - 无障碍输入框
- `accessible/AccessibleCard.ets` - 无障碍卡片
- `AccessibleComponents.ets` - 统一导出

**特性**:
- ✅ 支持老年模式（字体放大1.25倍）
- ✅ 最小触控区域44dp
- ✅ 高对比度模式
- ✅ 语音播报支持
- ✅ 符合WCAG 2.1 AA级标准

**使用方法**:
```typescript
import { AccessibleButton, AccessibleText, AccessibleCard } from '../components/AccessibleComponents';

// 无障碍按钮
AccessibleButton({
  text: '预约挂号',
  type: 'primary',
  fullWidth: true,
  voiceHint: '点击预约挂号按钮',
  onClick: () => {
    // 处理点击
  }
})

// 无障碍文本
AccessibleText({
  content: '健康报告',
  type: 'title'  // title | subtitle | body | caption
})

// 无障碍卡片
AccessibleCard({
  title: '预约挂号',
  subtitle: '快速预约医院科室',
  icon: '📅',
  showArrow: true,
  onClick: () => {
    // 处理点击
  }
})
```

### 3. 图表可视化组件

**位置**: `entry/src/main/ets/components/`

**文件列表**:
- `charts/HealthLineChart.ets` - 健康数据折线图
- `charts/HealthGauge.ets` - 健康指标仪表盘
- `charts/HealthRadarChart.ets` - 健康评分雷达图
- `ChartComponents.ets` - 统一导出

**特性**:
- ✅ Canvas绘制，性能优秀
- ✅ 支持老年模式
- ✅ 动态数据更新
- ✅ 响应式布局

**使用方法**:
```typescript
import { HealthLineChart, HealthGauge, HealthRadarChart } from '../components/ChartComponents';

// 折线图
HealthLineChart({
  data: [
    { x: 1, y: 120 },
    { x: 2, y: 125 },
    { x: 3, y: 118 }
  ],
  title: '血压趋势',
  unit: 'mmHg',
  showGrid: true,
  lineColor: '#1677FF'
})

// 仪表盘
HealthGauge({
  value: 85,
  maxValue: 100,
  title: '健康评分',
  unit: '分',
  warningThreshold: 70,
  dangerThreshold: 90
})

// 雷达图
HealthRadarChart({
  dimensions: [
    { name: '血压', value: 85, maxValue: 100 },
    { name: '血糖', value: 70, maxValue: 100 },
    { name: '心率', value: 90, maxValue: 100 }
  ],
  title: '健康维度分析'
})
```
