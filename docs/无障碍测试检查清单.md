# 无障碍测试检查清单

## 📋 测试概述

本检查清单用于验证星云医疗助手是否符合WCAG 2.1 AA级无障碍标准。根据项目规划V14.0 - 9.3提示词6要求编写。

## ✅ 测试检查项

### 1. 字体和文本测试

#### 1.1 字体大小
- [ ] **标准模式字体大小**
  - 标题 ≥ 18sp
  - 正文 ≥ 14sp
  - 辅助文字 ≥ 12sp
  - 按钮文字 ≥ 14sp

- [ ] **老年模式字体大小**
  - 标题 ≥ 24sp
  - 正文 ≥ 20sp
  - 辅助文字 ≥ 18sp
  - 按钮文字 ≥ 20sp

- [ ] **字体缩放**
  - 支持100%-200%缩放
  - 缩放后内容不丢失
  - 布局不破坏

#### 1.2 文本可读性
- [ ] **行间距**
  - 行高至少为字体大小的1.5倍
  - 段落间距至少为字体大小的2倍

- [ ] **字间距**
  - 字间距至少为字体大小的0.12倍
  - 词间距至少为字体大小的0.16倍

- [ ] **文本对齐**
  - 长文本左对齐
  - 标题可居中
  - 避免两端对齐（影响可读性）

### 2. 颜色和对比度测试

#### 2.1 对比度检查
- [ ] **普通文本对比度**
  - 正文与背景对比度 ≥ 4.5:1
  - 标题与背景对比度 ≥ 4.5:1
  - 按钮文字与背景对比度 ≥ 4.5:1

- [ ] **大文本对比度**
  - 18sp以上文本对比度 ≥ 3:1

- [ ] **UI组件对比度**
  - 图标与背景对比度 ≥ 3:1
  - 边框与背景对比度 ≥ 3:1

#### 2.2 颜色独立性
- [ ] **不依赖颜色传达信息**
  - 错误提示不仅用红色，还用图标/文字
  - 成功提示不仅用绿色，还用图标/文字
  - 警告提示不仅用黄色，还用图标/文字

- [ ] **高对比度模式**
  - 启用高对比度模式后所有文本清晰可读
  - 所有交互元素清晰可辨

### 3. 触控和交互测试

#### 3.1 触控区域
- [ ] **最小触控尺寸**
  - 所有按钮 ≥ 44x44dp
  - 所有图标按钮 ≥ 44x44dp
  - 所有列表项 ≥ 48dp高

- [ ] **老年模式触控区域**
  - 所有按钮 ≥ 48x48dp
  - 间距增大至12dp

#### 3.2 交互反馈
- [ ] **视觉反馈**
  - 点击有视觉变化（颜色、阴影）
  - 焦点有明确指示
  - 加载状态清晰

- [ ] **触觉反馈**
  - 关键操作有振动反馈
  - 可在设置中开关

- [ ] **语音反馈**
  - 屏幕阅读器模式下有语音提示
  - 操作成功有语音确认

### 4. 屏幕阅读器测试

#### 4.1 语义化标签
- [ ] **所有图片有alt属性**
  - 装饰性图片alt为空
  - 功能性图片alt描述功能
  - 复杂图片有详细描述

- [ ] **所有按钮有标签**
  - 图标按钮有aria-label
  - 按钮功能描述清晰
  - 状态变化有通知

- [ ] **所有输入框有标签**
  - 输入框有关联的label
  - placeholder不是唯一提示
  - 错误提示可被读取

#### 4.2 导航顺序
- [ ] **焦点顺序合理**
  - Tab键顺序符合视觉顺序
  - 跳过重复内容（如导航栏）
  - 无键盘陷阱

- [ ] **焦点可见**
  - 焦点有明确的视觉指示
  - 焦点样式对比度足够
  - 焦点不消失

### 5. 键盘导航测试

#### 5.1 基本导航
- [ ] **Tab导航**
  - 所有交互元素可通过Tab到达
  - Tab顺序合理
  - Shift+Tab可反向导航

- [ ] **快捷键**
  - Enter/Space激活按钮
  - Esc关闭对话框
  - 方向键在列表中导航

#### 5.2 特殊组件
- [ ] **下拉菜单**
  - 方向键选择选项
  - Enter确认选择
  - Esc关闭菜单

- [ ] **对话框**
  - 打开时焦点移到对话框
  - 关闭时焦点返回触发元素
  - 焦点限制在对话框内

### 6. 语音控制测试

#### 6.1 语音命令
- [ ] **基本命令**
  - "返回" - 返回上一页
  - "首页" - 返回首页
  - "帮助" - 显示帮助

- [ ] **功能命令**
  - "预约挂号" - 打开预约页面
  - "我的预约" - 查看预约列表
  - "健康数据" - 查看健康记录

#### 6.2 语音反馈
- [ ] **操作反馈**
  - 页面进入有语音提示
  - 操作成功有语音确认
  - 错误有语音警告

### 7. 老年模式测试

#### 7.1 视觉优化
- [ ] **字体增大**
  - 所有文字增大25%
  - 图标相应增大
  - 布局自适应

- [ ] **间距增大**
  - 页面边距增大
  - 组件间距增大
  - 按钮间距增大

#### 7.2 操作简化
- [ ] **流程简化**
  - 核心功能步骤减少
  - 提供快速操作入口
  - 智能默认值

- [ ] **提示增强**
  - 操作指引更详细
  - 错误提示更友好
  - 帮助信息更全面

### 8. 响应式布局测试

#### 8.1 不同设备
- [ ] **手机（< 600dp）**
  - 单列布局
  - 底部导航
  - 内容完整显示

- [ ] **平板竖屏（600-840dp）**
  - 双列布局
  - 侧边导航可选
  - 利用屏幕空间

- [ ] **平板横屏（> 840dp）**
  - 三列布局
  - 侧边导航
  - 多窗口支持

#### 8.2 横竖屏切换
- [ ] **布局自适应**
  - 切换后布局正确
  - 内容不丢失
  - 焦点位置合理

### 9. 性能测试

#### 9.1 响应速度
- [ ] **操作响应**
  - 点击响应 < 100ms
  - 页面切换 < 300ms
  - 列表滚动流畅（> 55fps）

- [ ] **无障碍功能**
  - 屏幕阅读器响应及时
  - 语音播报不卡顿
  - 触觉反馈无延迟

### 10. 兼容性测试

#### 10.1 辅助技术
- [ ] **屏幕阅读器**
  - TalkBack兼容
  - VoiceOver兼容（iOS）
  - 内容完整可读

- [ ] **外接设备**
  - 外接键盘支持
  - 外接鼠标支持
  - 开关控制支持
