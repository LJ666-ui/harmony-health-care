# 星云医疗助手 - 项目功能清单

## 一、用户认证与管理

### 1.1 用户注册与登录
- 手机号 + 密码注册，支持验证码验证

- Token 令牌自动管理，持久化存储

- 退出登录功能

### 1.2 个人中心
- 个人信息展示与编辑（姓名、年龄、性别、身高、体重、血型、过敏史等）
- 头像上传与更换
- 紧急联系人管理
- 浏览历史查看
- 我的收藏管理
- 隐私设置

### 1.3 管理员系统
- 管理员独立登录入口
- 管理员仪表盘（数据统计概览）
- 用户管理（用户列表、状态管理）
- 家庭成员管理
