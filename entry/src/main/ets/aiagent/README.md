# AI智能问诊系统

## 功能概述

医疗AI智能体全流程服务系统，提供多智能体会诊、RAG知识检索、多轮对话记忆等核心功能。

## 核心特性

### 1. 多智能体会诊
- **5个专科智能体**：内科、外科、中医、营养、康复
- **意图识别**：自动识别用户问题的医疗意图类型
- **智能分发**：根据意图分发给对应的专科智能体
- **协作回答**：多个智能体协作生成综合诊疗建议

### 2. RAG知识检索
- **向量检索**：基于向量相似度检索医学知识库
- **知识过滤**：过滤低相关度文档（相似度阈值0.7）
- **引用标注**：标注回答所引用的知识来源

### 3. 多轮对话记忆
- **上下文管理**：存储和管理对话历史
- **窗口限制**：最多保留10轮对话
- **会话隔离**：不同用户的对话相互独立

### 4. 意图分类
- **7种意图类型**：症状咨询、用药指导、康复建议、营养咨询、中医咨询、外科咨询、一般咨询
- **实体提取**：提取症状、疾病、药物、身体部位等医疗实体

## 使用方式

### 1. 从首页进入

在应用首页的功能卡片中，点击"AI智能问诊"卡片即可进入。

### 2. 问诊流程

1. **输入问题**：在输入框中描述您的症状或问题
   - 例如："我最近头痛"
   - 例如："高血压怎么管理"
   - 例如："术后康复饮食要注意什么"

2. **智能识别**：系统自动识别问题意图并分发给对应专科智能体

3. **生成建议**：智能体基于医学知识生成专业诊疗建议

4. **查看结果**：查看多个智能体的会诊建议和知识引用

### 3. 多轮对话

- 支持追问和澄清
- 系统会记住上下文，提供连贯的对话体验

## 专科智能体介绍

### 1. 内科智能体 (InternalMedicineAgent)
- **专长**：慢性病管理、症状分析、常见病诊断建议
- **适用场景**：感冒、高血压、糖尿病、冠心病等内科疾病

### 2. 外科智能体 (SurgeryAgent)
- **专长**：外伤处理、手术咨询、伤口护理
- **适用场景**：外伤、手术前后咨询、伤口护理

### 3. 中医智能体 (TCMAgent)
- **专长**：中药调理、针灸指导、体质辨识
- **适用场景**：中医调理、体质辨识、养生保健

### 4. 营养智能体 (NutritionAgent)
- **专长**：饮食建议、食疗方案、营养素补充
- **适用场景**：健康饮食、疾病食疗、营养指导

### 5. 康复智能体 (RehabAgent)
- **专长**：运动处方、康复训练、功能恢复
- **适用场景**：术后康复、运动指导、功能恢复

## API接口

### 问诊接口

```typescript
import { MultiAgentOrchestrator } from '../aiagent';

// 创建编排器实例
const orchestrator = new MultiAgentOrchestrator(
  intentClassifier,
  memory,
  ragRetriever
);

// 发起问诊请求
const response = await orchestrator.orchestrate({
  sessionId: 'session_001',
  question: '我最近头痛',
  userId: 'user_001'
});

// 获取会诊结果
console.log(response.answers);  // 智能体回答列表
console.log(response.summary);  // 会诊总结
```

## 注意事项

1. **仅供参考**：AI提供的诊疗建议仅供参考，不能替代专业医生的诊断
2. **紧急情况**：如有紧急医疗情况，请立即就医或拨打急救电话
3. **用药建议**：具体用药请遵医嘱，不要自行调整用药剂量
4. **隐私保护**：对话记录仅用于提供更好的服务，不会泄露给第三方
