Step 4: 替换aboutToAppear()方法，改为从后端加载数据:

```typescript
@State herbalList: HerbalMedicine[] = []
@State isLoading: boolean = false
@State currentPage: number = 1
@State hasMore: boolean = true
@State keyword: string = ''
@State total: number = 0     // 总数(用于显示"共730种")

async aboutToAppear() {
  console.info('[HerbalListPage] 页面加载，开始获取药材数据')
  await this.loadHerbalList(true)
}

async loadHerbalList(isRefresh: boolean = false) {
  this.isLoading = true
  
  if (isRefresh) {
    this.currentPage = 1
    this.hasMore = true
  }
  
  try {
    let url = `${HERBAL_API.PAGE}?page=${this.currentPage}&size=20`
    if (this.keyword && this.keyword.trim() !== '') {
      url += `&keyword=${encodeURIComponent(this.keyword.trim())}`
    }
    
    const res = await HttpUtil.get(url)
    
    if (res.code === 200 && res.data) {
      const pageResult = res.data as {
        records: HerbalMedicine[]
        total: number
        pages: number
        current: number
      }
      
      if (isRefresh) {
        this.herbalList = pageResult.records
      } else {
        this.herbalList = [...this.herbalList, ...pageResult.records]
      }
      
      this.total = pageResult.total
      this.hasMore = this.currentPage < pageResult.pages
      this.currentPage++
    }
  } catch (error) {
    console.error('[HerbalListPage] 加载失败:', error.message)
    ToastUtil.showShort('网络错误')
  } finally {
    this.isLoading = false
  }
}
```

> **技术选型**: HarmonyOS RelationalStore (@ohos.data.relationalStore)
> **数据库位置**: `entry/src/main/ets/database/local.db`
> **设计原则**: 离线优先(Offline-First) — 即使没网，核心数据也能用

### 3.2 你要干什么

创建8张本地表：
1. local_user_cache - 用户登录缓存（"记住我"功能）
2. offline_health_record - 离线健康记录队列
3. local_article_cache - 文章阅读缓存
4. local_device_cache - 设备配对缓存
5. sync_queue - 数据同步队列
6. **local_browse_history** - 浏览历史记录 ⭐核心新表
7. **local_search_history** - 搜索历史
8. **local_collection_cache** - 收藏夹缓存

### 3.3 创建了什么文件

**新建**:
- `database/RdbHelper.ets` - 数据库单例工具类
- `database/DatabaseConstants.ets` - 表名/字段名常量
- `database/init.sql` - 建表SQL脚本（8张表）
- `utils/SyncManager.ets` - 数据同步管理器
- `utils/BrowseTracker.ets` - 浏览行为追踪器
- `models/LocalModels.ets` - 本地实体类（8个Model）

**修改**:
- `EntryAbility.ets` - 在onCreate()中调用RdbHelper.init()
