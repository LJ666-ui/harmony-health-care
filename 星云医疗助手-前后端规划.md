## 一、项目概述

### 1.1 项目定位

**星云医疗助手**是一款基于HarmonyOS原生开发的分布式智慧医疗全周期管理应用，聚焦老年群体就医、康复、健康管理的核心痛点，打造"院前-院中-院后-日常科普"全场景闭环服务。

### 1.2 核心价值

| 痛点 | 解决方案 | 价值体现 |
|------|----------|----------|
| 老龄化社会医疗需求 | 全流程智能服务 | 提升就医效率50% |
| 医疗资源分布不均 | 分布式数据协同 | 打破数据孤岛 |
| 就医流程繁琐 | 智慧导航+AI规划 | 减少等待时间 |
| 康复管理缺失 | AI共病评估+3D指导 | 提升康复效果 |
| 健康知识匮乏 | 场景化科普学习 | 增强自我管理能力 |

### 5.3 爬虫实现

#### 5.3.1 医疗科普爬虫

```python
class MedicalArticleSpider(scrapy.Spider):
    name = 'medical_article'
    start_urls = ['https://www.example.com/medical/articles']
    
    def parse(self, response):
        articles = response.css('.article-item')
        for article in articles:
            yield {
                'title': article.css('.title::text').get(),
                'content': article.css('.content::text').get(),
                'category': article.css('.category::text').get(),
                'source': 'example.com',
                'publish_time': article.css('.time::text').get()
            }
        
        next_page = response.css('.next-page::attr(href)').get()
        if next_page:
            yield response.follow(next_page, self.parse)
```

#### 5.3.2 医院信息爬虫

```python
class HospitalSpider(scrapy.Spider):
    name = 'hospital'
    start_urls = ['https://www.example.com/hospitals']
    
    def parse(self, response):
        hospitals = response.css('.hospital-item')
        for hospital in hospitals:
            yield {
                'name': hospital.css('.name::text').get(),
                'level': hospital.css('.level::text').get(),
                'address': hospital.css('.address::text').get(),
                'phone': hospital.css('.phone::text').get(),
                'departments': [dept.css('::text').get() for dept in hospital.css('.departments li')]
            }
        
        next_page = response.css('.next-page::attr(href)').get()
        if next_page:
            yield response.follow(next_page, self.parse)
```
