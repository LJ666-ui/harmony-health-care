## 一、项目概述

### 1.1 项目定位

**星云医疗助手**是一款基于HarmonyOS原生开发的分布式智慧医疗全周期管理应用，聚焦老年群体就医、康复、健康管理的核心痛点，打造"院前-院中-院后-日常科普"全场景闭环服务。

### 1.2 核心价值

| 痛点 | 解决方案 | 价值体现 |
|------|----------|----------|
| 老龄化社会医疗需求 | 全流程智能服务 | 提升就医效率50% |
| 医疗资源分布不均 | 分布式数据协同 | 打破数据孤岛 |
| 就医流程繁琐 | 智慧导航+AI规划 | 减少等待时间 |
| 康复管理缺失 | AI共病评估+3D指导 | 提升康复效果 |
| 健康知识匮乏 | 场景化科普学习 | 增强自我管理能力 |

### 1.3 目标用户

- **患者/老年用户**：需要就医导航、康复管理、健康监测
- **家属**：远程关注患者健康状态、接收预警通知
- **医护人员**：快速获取患者数据、AI辅助诊断

### 1.4 项目形态

- 支持手机、平板、大屏、手表多端适配
- 可直接参赛演示的完整Demo
- 功能模块完整且可流畅运行

## 四、核心功能模块

### 4.1 模块一：分布式跨院数据协同

#### 功能描述
实现跨院病历/检查报告无感同步，转院线上审批，数据权限分级管控。

#### 核心功能
| 功能 | 描述 | 技术实现 |
|------|------|----------|
| 转院申请 | 线上提交与审批 | 分布式数据同步 |
| 病历同步 | 无需重复录入 | 分布式软总线 |
| 权限管控 | 医生/家属/患者分级 | 角色权限系统 |
| 数据脱敏 | 敏感信息自动隐藏 | 端侧加密计算 |
| 操作追溯 | 全程日志记录 | 审计日志系统 |

#### 用户场景
1. 患者在A医院检查后转院到B医院
2. B医院医生一键获取患者完整病历
3. 家属可查看脱敏后的健康概况
4. 患者可随时撤销数据授权

### 4.2 模块二：智慧就医导航系统

#### 功能描述
提供院内AR实景导航、智慧停车、就医流程智能规划。

#### 核心功能
| 功能 | 描述 | 技术实现 |
|------|------|----------|
| 智慧停车 | 车位查询/预约/导航/缴费 | Map Kit + Location Kit |
| 院内导航 | AR实景指引 | AR Engine + 3D渲染 |
| 科室搜索 | 快速定位目标科室 | POI检索 |
| 路径优化 | 最短路线规划 | 智能算法 |
| 就医规划 | 结合挂号时间排序 | AI调度 |

#### 用户场景
1. 用户到达医院前预约停车位
2. 导航直达车位，反向寻车
3. AR导航指引到目标诊室
4. 系统智能排序检查流程

### 4.3 模块三：AI共病康复管理

#### 功能描述
健康数据录入监测，端侧AI共病风险评估，3D康复动作指导。

#### 核心功能
| 功能 | 描述 | 技术实现 |
|------|------|----------|
| 数据录入 | 手动/语音录入健康数据 | 语音识别 + 表单 |
| 风险评估 | 三高+跌倒+肌少症联合预测 | MindSpore Lite |
| 3D指导 | 康复动作演示与纠正 | 3D模型 + AI姿态识别 |
| 异常预警 | 多端同步预警通知 | 分布式消息 |
| 方案定制 | 个性化康复方案 | AI推荐引擎 |

#### 共病评估模型
```
输入数据：
- 血压、血糖、体重、步态、心率

AI模型：
- 肌少症-衰弱联合预测
- 三高-跌倒风险关联分析
- 多病症交叉影响评估

输出结果：
- 风险等级（低/中/高）
- 预警建议
- 康复方案推荐
```

    avatar VARCHAR(255) COMMENT '头像URL',
    role TINYINT DEFAULT 0 COMMENT '角色：0患者 1家属 2医生 3护士',

### 12.4 网络请求指南

```typescript
import http from '@ohos.net.http'

interface HttpParams {
  url: string
  method?: http.RequestMethod
  data?: object
  header?: object
}

export async function httpRequest(params: HttpParams): Promise<object> {
  const httpRequest = http.createHttp()
  
  try {
    const response = await httpRequest.request(params.url, {
      method: params.method || http.RequestMethod.GET,
      extraData: params.data,
      header: params.header || { 'Content-Type': 'application/json' }
    })
    
    if (response.responseCode === 200) {
      return JSON.parse(response.result as string)
    }
    throw new Error('请求失败')
  } finally {
    httpRequest.destroy()
  }
}
```
