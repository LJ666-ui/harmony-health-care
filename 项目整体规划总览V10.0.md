### 模块1: 用户认证与隐私安全体系

**模块概述**: 三级隐私保护体系（隐私政策→设置中心→高级设置），符合GDPR和个人信息保护法要求。

| 页面 | 功能描述 | 技术方案 | 状态 | 验收标准 |
|------|----------|----------|------|----------|
| Index.ets | 启动品牌展示，2秒跳转登录 | @Entry + setTimeout路由 | ✅ | "星云医疗助手"+slogan |
| Login.ets | 用户名密码登录+JWT获取 | POST /user/login + Token缓存 | ✅ | 用户名/手机号登录 |
| Register.ets | 表单验证(3-20位)+BCrypt加密注册 | @Valid + POST /user/register | ✅ | 密码强度指示器 |
| PrivacyPage.ets | 授权设备/系统权限/访问日志管理 | Privacy API + RDB缓存 | ✅新增 | 手环/血压计授权列表 |
| PrivacySettingPage.ets | 精细隐私开关/授权项目/审计日志 | 三Tab布局 | ✅新增 | 位置/健康/浏览/诊断独立开关 |
| PrivacyPolicyPage.ets | V1.0隐私政策全文+同意按钮 | LocalStorage状态检查 | ✅新增 | 数据范围/使用方式/权利说明 |

**核心功能点 (8个)**:
- F01-01: JWT双Token机制（AccessToken 2h + RefreshToken 7d）
- F01-02: 登录状态持久化（local_user_cache表 + AppStorage全局状态）
- F01-03: 隐私政策强制阅读（首次启动拦截 → 未同意无法使用）
- F01-04: 设备OAuth2授权（手环/血压计/血糖仪授权流程）
- F01-05: 权限粒度控制（相机/存储/位置/蓝牙独立开关）
- F01-06: 数据访问审计（谁/何时/访问了什么/做了什么操作）
- F01-07: 数据导出与注销（一键导出PDF/删除个人数据，符合"被遗忘权"）
- F01-08: 多因素认证预留（指纹/人脸/短信验证码接口预留）

**对应后端API (9个)**:
`POST /user/register` | `POST /user/login` | `GET /user/info` | `POST /user/update`

### 6.3 各新增模块技术亮点

#### 模块A: AI Agent多代理系统 (aiagent/)

```
MedicalAgent (抽象基类)
  ├── RAGRetriever: 10篇医学知识库文档检索增强
  │   └── 内科: 高血压预防控制 / 糖尿病饮食管理
  │   ├── 外科: 外伤处理原则 / 手术前后注意事项
  │   ├── 中医: 体质辨识调理 / 常用中药材功效
  │   ├── 营养: 老年人营养需求 / 常见疾病食疗
  │   └── 康复: 康复训练原则 / 运动处方制定
  ├── ConversationMemory: 短期/长期记忆管理
  └── SpecialistAgents: 内科/外科/中医/营养/康复 5专家代理协作

处理流程: 用户提问 → RAG检索相关知识 → 构建对话历史 → 组装Prompt 
        → 生成响应 + 计算置信度 → 低置信度建议咨询真人 → 记录记忆
```

#### 模块B: 实时健康监控 (monitoring/)

- **三策略融合**: 阈值规则(医学标准) + Z-Score统计偏移(个人基线) + 趋势突变(时序变化率)
- **五级预警**: NORMAL→LOW→MEDIUM→HIGH→CRITICAL 逐级升级处理
- **覆盖指标**: 血压(收缩/舒张)、血糖(空腹/随机/餐后)、心率、体重、体温、血氧(预留)
- **后台运行**: 单例RealtimeMonitor，默认5分钟间隔，可配置1min~1h
- **紧急响应**: CRITICAL级自动拨打紧急联系人+发送SMS含GPS位置+APP最强提醒

#### 模块C: 智慧停车 (parking/)

- **完整闭环**: 推荐→预约→计时→拍照记录→就诊→AR找车→计费→支付→发票
- **可视化**: ParkingMap Canvas绘制车位格子图，实时状态着色
- **AR导航**: NavigationEngine AR实景导航从医院出口到停车位
- **灵活计费**: 免费时长/小时费率/日封顶/最大天数全部可配置
- **就医联动**: 预约成功自动推荐附近停车场；就诊结束自动触发找车

#### 模块D: 分布式协同 (distributed/)

- **4级权限**: DOCTOR_FULL(完全) / DOCTOR_PARTIAL(部分脱敏) / FAMILY(完全脱敏) / SELF(无脱敏)
- **脱敏规则**: 正则匹配替换（姓名首字保留/身份证前3后4/手机前3后4/地址省市保留）
- **AES-256-GCM**: 端到端加密，写入SecurityLevel.S1最高安全级别KVStore
- **自动同步**: 鸿蒙分布式能力，跨设备(PHONE/TABLET/WATCH/终端)自动同步
- **授权生命周期**: 创建→生效(active)→使用(记录日志)→过期(expired)→可撤销(revoked)

#### 模块E: 医疗影像AI (medical/)

- **5种影像**: CHEST_XRAY / CT_SCAN / SKIN_LESION / RETINA / GENERAL
- **分析结果**: findings[]病灶列表(标签/置信度/bbox/严重程度/描述) + overallDiagnosis + urgencyLevel
- **AnnotationOverlay**: HIGH红色3px / MEDIUM橙色2px / LOW绿色1px 边界框可视化
- **双模式**: Mock模式(5组典型病例预设数据) + 真实模式(预留AI模型接口)

#### 模块F: 增强AI引擎 (ai/)

- **10维特征向量**: age/gender/bmi/BPsys/BPdia/fastingBS/heartRate/sleepHours/exerciseMin/smokeDrink
- **5大慢性病**: 高血压/糖尿病/跌倒风险/衰弱综合征/肌少症
- **混合模式**: AI_MODEL(需网) / RULE_BASED(离线) / HYBRID(推荐)
- **RuleBasedAssessor**: 200+条医学指南规则(WHO/ADA等)
- **ReportGenerator**: 7章节专业报告(信息/评分/分析/解读/建议/趋势/行动)
- **TrendAnalyzer**: improving/stable/declining 三种趋势 + 各疾病变化方向幅度
